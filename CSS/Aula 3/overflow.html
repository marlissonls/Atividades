Caixas com exibição Overflow

Usando a propriedade 'overflow' com valor 'visible', o texto escrito será totalmente mostrado mesmo que não caiba dentro do tamnha delimitado.

Usando o 'overflow' com valor 'hidden', o texto escrito não será mostrado completo caso ele não caiba no tamanho delimitado.

Usando o 'overflow' com valor scroll', o conteúdo pode ser totalmente visualizado mesmo que não caiba dentro do tamanho delimitado, basta rolar a barra para exibir o conteúdo ocultado.

Usando o 'overflow' no modo 'auto', o texto será adaptado para que todo ele seja visível dentro do espaço delimitado.
