Nesta página, vamos exemplificar o uso do CSS nas forma básicas: Inline, Internal e External

1. Nos 3 ítens abaixo foram aplicados estilos de cor. Cada estilo foi aplicado segundo a própria legenda:

  • Inline, usando o atributo style="red" na tag <li>.
  • Internal, através do atributo id="internal" e usando a tag <style>.
  • External, através do atributo id="external" e usando a tag <link> que anexou o arquivo style.css.

2. No ítem abaixo serão aplicadas as cores red, green e blue cada uma delas em uma das 3 formas de aplicação de CSS (Inline, Internal e External, respectivamente) para se verificar a prioridade de aplicação dessas 3 formas de aplicação.

  • um elemento de texto

Como esperado, a forma Inline teve prioridade sobre as formas Internal e External
